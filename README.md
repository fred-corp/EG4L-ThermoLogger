# EG4L : ThermoLogger

## Project specs

* Thermocouple inputs :
  * 2x "Generic" MAX6675 front-end (SPI)
  * 2x "Custom" front-end (LTC1049 Amplifier, MAX1167 DAC (SPI))
* Interface :
  * 1602 Screen (OLED, LCD) (SPI/Parallel)
  * 4-way joypad (GPIO)
  * USB (Serial) (Programming, GPIB?)
* Storage :
  * SD Card (SPI)
* Power :
  * AME25-5SVZ Power supply (5V, 4.1A)
